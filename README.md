# DAWRV - Digital Audio Workstation Response to Voice

**Featuring RHEA - Responsive Heuristic Environment Assistant**

## Quick Start

Run DAWRV:
```
cd ~/DAWRV-Project
npm start
```

Development mode (with DevTools):
```
npm run dev
```

## Voice Commands

- "Play" - Start playback
- "Stop" - Stop playback  
- "Record" - Start recording
- "Undo" - Undo last action
