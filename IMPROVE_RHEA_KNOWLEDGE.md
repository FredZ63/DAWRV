## Technical Notes

### How the Knowledge Base Works
1. **Chunking** - Text is split into ~500 word chunks
2. **Embedding** - Each chunk gets a vector representation
3. **Storage** - Stored locally in browser (IndexedDB)
4. **Retrieval** - When you ask a question, relevant chunks are found
5. **Context** - Retrieved chunks are added to the AI's context

### Keeping Knowledge Fresh
- Knowledge persists across sessions
- Export regularly as backup
- Clear and re-import if needed
- Add new knowledge as you learn
