## Tips

1. **Natural Language**: You can say commands in natural ways - "play the project", "stop playback", etc.
2. **Multiple Variations**: Each command has multiple keyword variations - use what feels natural
3. **Feedback**: RHEA will confirm each command with a spoken response
4. **Cooldown**: Commands have a 4-second cooldown to prevent accidental repeats

## Quick Reference Card

**Most Common Commands:**
- "play" / "stop" - Transport control
- "record" - Start recording
- "undo" / "redo" - Undo/redo
- "save" - Save project
- "new track" - Create track
- "mute" / "solo" - Track control
- "zoom in" / "zoom out" - Navigation
- "loop" - Toggle loop
